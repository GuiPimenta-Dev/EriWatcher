# EriWatcher

## Este projeto foi desenvolvido com intuito de automatizar tickets de inclusao/alteraçao/exclusao de usuários no Banco de Dados, onde um robo vigia a fila de chamados e ao perceber um chamado específico automaticamente atende o chamado baixando uma planilha criada especificamente para a automaçao, onde a própria planilha cria as querys de acordo com as informacoes inseridas. As querys geradas pela planilha sao inseridas no banco de dados e todos os passos sao logados em um chatbot via Telegram.
